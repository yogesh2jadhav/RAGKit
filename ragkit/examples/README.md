# RAGKit Examples

Welcome to the RAGKit learning path.

The examples are intentionally numbered and should be executed in order.

Each example introduces **one new concept** while building towards a complete Retrieval-Augmented Generation (RAG) application.

---

## Learning Path

| Example | Topic |
|---------:|-------|
| 01 | Load Documents |
| 02 | Chunk Documents |
| 03 | Generate Embeddings |
| 04 | Build Vector Index |
| 05 | Similarity Search |
| 06 | Retrieval Pipeline |
| 07 | Document Indexer |
| 08 | Transformers |
| 09 | Rerankers |
| 10 | Interactive Chat |
| 11 | Complete RAG Application |

---

## Philosophy

Each example teaches one concept.

Avoid skipping examples.

Every subsequent example builds on knowledge learned previously.
